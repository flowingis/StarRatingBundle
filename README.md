StarRatingBundle
================

**StarRatingBundle** is a Symfony Bundle for easy star rating system.

## Installation

* Install the bundle via Composer (?)
* Include within the AppKernel
* Add the bundle to assetic in config.yml

    assetic:
        bundles:        [ IdeatoStarRatingBundle ]

* Add js and css
